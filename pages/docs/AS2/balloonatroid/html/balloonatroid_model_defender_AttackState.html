execute

function execute (
sm:IStateMachine) : Void

The logic to be executed when the state is updated.

Moves the defender based on current speed and direction.
Broadcasts DEFENDER_IS_DEAD notification if defender is off screen.
Broadcasts GO_DEFENDER_HIT notification to transition
to the hit state if the hitBySprite property is set.
