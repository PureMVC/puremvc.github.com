NAME

org::puremvc::perl5::multicore::utilities::statemachine::State

Defines a state in a PureMVC application.

DESCRIPTION

A finite state machine State object.

  • is identified by its name.

  • Optionally defines PureMVC Notification names to be sent by the StateMachine when

    • the StateMachine current state is changed to this one.

    • the StateMachine current state is this one and is about to enter it.

    • the StateMachine current state is this one and is about to exit it.

  • Optionally defines transitions to other states (target states) for a given action.
