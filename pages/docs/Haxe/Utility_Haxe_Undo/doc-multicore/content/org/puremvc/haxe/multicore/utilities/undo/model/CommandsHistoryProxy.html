class org.puremvc.haxe.multicore.utilities.undo.model.CommandsHistoryProxy
extends org.puremvc.haxe.multicore.patterns.proxy.Proxy
implements org.puremvc.haxe.multicore.interfaces.IProxy

The model that keeps track of the commands. It provides methods to get the next or the previous command from the history.

In order to record a command into the history, you must set the type of its notification to UndoableCommandTypeEnum.RECORDABLE_COMMAND
