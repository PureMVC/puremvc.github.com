Packageorg.puremvc.as3.utilities.startupmanager.assetloader.controller
Classpublic class LoadAssetGroupInstructions

LoadAssetGroupInstructions defines the instructions required by the LoadAssetGroupCommand, sent to it as the notification body. For the instructions to be sufficient, there must be a groupId and a groupOfUrls.

See StartupForAssets demo for an example use of this class.
