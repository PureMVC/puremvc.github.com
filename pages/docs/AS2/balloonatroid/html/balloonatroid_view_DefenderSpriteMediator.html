DefenderSpriteMediator

Kind of class:class
Inherits from:Mediator
Implements:
  • IMediator
Classpath:balloonatroid.view.DefenderSpriteMediator
File last modified:Monday, 17 March 2008, 16:04:00
A Mediator for interacting with the Defender Sprite.
Adds and removes itself from the sprite entities collection
on onRegister and onRemove.
Registers EnterFrame handler and updates its current state.

function onRegister (
) : Void

Called by the View when the Mediator is registered.
Registers EnterFrame handler.
Adds defender balloon to sprite entities collection.

  • Expects DefenderProxy and SpriteEntitiesProxy to be registered previously.

onRemove

function onRemove (
) : Void

Called by the View when the Mediator is removed.
